Page padding 20–28px · card gaps 8–12px · section gaps 16–24px. Borders are consistently 1–1.5px solid white at 12–20% opacity. Backdrop blur ranges 8px (inputs) to 18px (hero cards). Shadows are used sparingly: device shell 0 24px 60px -20px rgba(0,0,0,.6); toast 0 8px 20px -6px rgba(0,0,0,.5).
Iconography
Single chip glyph, tinted violet/cyan via CSS filter (saturate 1.6, hue-rotate 220deg, brightness 1.3). No other custom icons or emoji are used — actions are plain-text buttons or single-letter avatars (initials).
